# Simple To-Do List Web Application

A clean, responsive, and user-friendly **To-Do List Application** built using **HTML, CSS, and JavaScript**.  
This project allows users to manage their daily tasks easily with features like adding, editing, deleting, completing, and filtering tasks.  
All tasks are saved automatically in the browser using **localStorage**.
